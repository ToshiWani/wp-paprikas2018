Plugin Dependencies
----------------------------

- Food and Drink Menu
- Instagram Feed

Add Pages
----------------------------
- Home
- Photos
- Press
- Contact Us

Add Menu (food)
----------------------------
Setup menu sections (e.g. appetizers and desserts)
- Menus > Menu Sections

Setup individual menus
- Menus > Menu Items

Setup Menus
- Dinner
- Lunch

NOTE: Make sure select correct template


Setup Homepage
----------------------------
Settings > Reading > A Static Page


Navigation
----------------------------
Appearance > Menus
- Add navigation in header, footer, and mobile
- Upload and set menu icons


Custom Fields
----------------------------
Configurations:
- Name: Page Fields
- Field Label: Headline
- Field Name: headline
- Field Type: WYSIWYG Editor
- Rules: (Post Type) + (is equal to) + (page)
- Position: High
